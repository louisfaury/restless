## Manual install

### Create and activate virtualenv
```
python -m venv .venv/
source .venv/bin/activate
```

### Install requirements
```
pip3 install -r requirements.txt
pip3 install -r test-requirements.txt
```

### Install library
```
pip3 install -e .
```
